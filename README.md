# Felipe Ivanaga - personal portfolio website

[Access website](https://ivanaga.vercel.app/)

Built using Nextjs, Tailwind, Pnpm, hosted on Vercel.

# Running locally

Make sure your have [pnpm](https://pnpm.io/) installed.

Available commands:

```bash
pnpm run dev
pnpm run build
pnpm run start
pnpm run lint
```
